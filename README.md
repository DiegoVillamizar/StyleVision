## Descripción del Proyecto
StyleVision es un aplicativo desarrollado en Python orientado al sector del estilismo y la barbería. Utiliza técnicas de visión artificial y procesamiento de imágenes para permitir a los usuarios visualizar cómo les quedaría un corte de cabello específico a partir de una fotografía propia.

## Definición del Problema
Los clientes a menudo enfrentan incertidumbre al elegir un nuevo corte de cabello por la incapacidad de visualizar el resultado final. Este proyecto resuelve esa necesidad aplicando transformaciones de imágenes para adaptar estilos predefinidos a la fisionomía del usuario.
